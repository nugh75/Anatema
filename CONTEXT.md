## Quick Reference
- **Stack**: Python (Flask), SQLite/SQLAlchemy, Bootstrap 5, Docker
- **Entry point**: `docker compose up -d` o `python app.py` (porta 5000)
- **Test**: manuale

## Domain
Sistema web per l'etichettatura collaborativa di risposte testuali estratte da file Excel. Interfaccia Bootstrap 5 con dashboard analytics. Supporta integrazione AI per suggerimenti di etichettatura.

### Key Files
- `app.py` — backend Flask
- `Dockerfile`, `docker-compose.yml` — containerizzazione
